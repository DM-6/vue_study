## awesomePOS 快餐店收银系统

#### 技术栈
Vue + Vue-router + Webpack + Element + Axios 

#### 开发工具
VSCode + MockPlus + iconFont  

使用MockPlus将大致的界面设计出来，如下图：
![快餐店收银系统设计图](https://user-gold-cdn.xitu.io/2018/8/7/16513f7b0d18e570?w=800&h=484&f=jpeg&s=75222)

页面基本布局：
- 左边：导航栏组件
- 右边：商品列表 常用商品列表
- 中间：订单列表
